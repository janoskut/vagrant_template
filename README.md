# Vagrant Templates

## Vagrant Basics

```
# Startup
vagrant up

# Connect
vagrant ssh

# Destroy machine without confirmation prompt
vagrant destroy -f

# Provision
vagrant reload --provision

# Stop
vagrant halt
```

## Features

- GIT
- Python (2/3), Pip
- VIM
  - Copy local .vimrc to machine
- EXA
  - Alias ls es exa
- ZSH
  - oh-my-zsh
  - Agnoster theme
